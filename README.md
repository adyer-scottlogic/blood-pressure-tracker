# Blood Pressure Tracker

A lightweight, privacy-first web app to track your blood pressure throughout the day with reminders.

## Features

- **Log readings** — Record systolic, diastolic, pulse, arm, date/time, and notes
- **Dashboard** — View latest stats, 7-day averages, and interactive trend chart
- **History** — Full reading history with filters by date range and BP category
- **Reminders** — Schedule up to 10 daily reminders with browser notifications and optional alarm sound
- **Export** — Download all readings as CSV
- **Privacy** — All data stored locally in browser localStorage — nothing sent to servers

## BP Categories (AHA)

| Category | Range |
|----------|-------|
| Normal | < 120 / < 80 |
| Elevated | 120–129 / < 80 |
| High Stage 1 | 130–139 / 80–89 |
| High Stage 2 | ≥ 140 / ≥ 90 |
| Crisis | > 180 / > 120 |

## Usage

1. Open `index.html` in any modern web browser
2. Navigate to **Add** tab to log a reading
3. Check **Dashboard** for trends and stats
4. Go to **Reminders** to set up daily notifications
5. Use **History** to review past readings and export data

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).

**Note:** Notifications require browser permission.

## Installation

No installation needed — just open the HTML file. All data is stored locally in your browser.
